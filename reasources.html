Resources

There are many reasources you can use to learn more about software engineering. There are online platforms like Coursera, Udemy, and Codecademy to help with coding skills. Frontend Masters (advanced JS/frontend) and Egghead.io (short, practical lessons) can be used for extra training. Also, OSSU (Open Source Society University) and Teach Yourself Computer Science can help with the basics to get you started. You can use LeetCode, HackerRank, and NeetCode to pratice various concepts you learn.
